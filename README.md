# Smart Lender
## Overview
Smart Lender is a smart loan management platform developed as part of the SkillWallet project. It helps users apply for loans, track repayments, and receive intelligent loan recommendations based on their financial profile.
## Problem Statement
Many people face difficulties in managing loans and understanding their repayment schedules. Smart Lender provides a simple and intelligent solution for loan management.
## Features
- User Registration and Login
- Loan Application System
- Loan Eligibility Checker
- EMI Calculator
- Loan Approval Tracking
- Repayment Management
- Dashboard for Users
- Secure Data Handling
## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Python / Node.js
- Database: MySQL
- Version Control: Git & GitHub
